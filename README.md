# Hotel Booking Analytics

📊 End-to-end analytics project exploring hotel booking demand, cancellations, and revenue insights using SQL, Python
and Power BI.

## Highlights
- Cleaned and structured hotel booking dataset with SQL
- Performed EDA and feature engineering in Python (seasonality, lead time, customer type)
- Built interactive Power BI dashboards showing:
  - Occupancy trends by season
  - Cancellation patterns by deposit type & lead time
  - ADR comparison: City vs Resort hotels
  - Guest demographics and revenue impact

## Tech Stack
- SQL (MySQL Workbench)
- Python (pandas, seaborn, matplotlib)
- Power BI (DAX measures, dashboards)
- GitHub (documentation + portfolio showcase)

## Insights
- Resort hotels show higher cancellation rates compared to city hotels
- Non-refundable deposits reduce cancellations significantly
- Families and couples dominate guest demographics
